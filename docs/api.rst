API documentation
=================

This API documentation is based on the source code of version |release| of the
`proc` package. The following generic modules are available:

- :py:mod:`proc`
- :py:mod:`proc.core`
- :py:mod:`proc.tree`

The following modules are based on the modules above:

- :py:mod:`proc.apache`
- :py:mod:`proc.cron`

These modules are meant to actually be used (for those who are interested in
them) but they also function as examples of how to use the :py:mod:`proc.core`
and :py:mod:`proc.tree` modules.
